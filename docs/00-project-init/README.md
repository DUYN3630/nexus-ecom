# 🚀 Nexus E-commerce — Project Overview

**Nexus E-commerce** is a modern e-commerce platform built with the MERN stack (MongoDB, Express.js, React, Node.js). 
It features a full product catalog, shopping cart, order management system, and an integrated IT Hub where users can book appointments with technical experts for device repairs.

## Core Features
- **User Authentication:** JWT-based secure login and registration.
- **Product Catalog & Management:** Browse products, categories, and reviews. Admin panel for inventory management.
- **Shopping Cart & Checkout:** Seamless purchasing flow.
- **IT Hub:** Book repair appointments with experts, manage repair requests.
- **AI Integration:** Google Gemini API powered assistant to help users with tech queries and product recommendations.
